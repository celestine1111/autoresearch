## How It Works

1. User selects **Category** (e.g. "Animals & Pets") and **Country** (e.g. "Australia")
2. Optimize All → Tavily search uses `include_domains` = AU animals + global animals
3. For AU + Animals: `rspca.org.au, apvma.gov.au, sydney.edu.au, abc.net.au, csiro.au, mindiampets.com.au` + `ncbi.nlm.nih.gov, nature.com, petmd.com, merckvetmanual.com`
4. If < 2 results found: falls back to unrestricted Tavily search (with substantive language filter)
5. Quotes extracted from raw page content of matched pages

## Adding/Editing Domains

Edit `includes/Content_Injector.php::get_authority_domains()` and update this file to match. Both must stay in sync.
